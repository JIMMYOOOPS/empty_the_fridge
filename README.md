## Description
Clean the fridge is an app powered by AI to help you manage whats left in your fridge. It suggests recipes based on the food you have and help you eat a varity of food you have never imagined.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
